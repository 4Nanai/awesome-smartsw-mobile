# Awesome Smart-Switch Mobile (ASM)

## How to run
1. clone repo
```shell
git clone git@github.com:4Nanai/awesome-smartsw-mobile.git && cd awesome-smartsw-mobile
```
2. install packages
```shell
npm install
```
3. copy env file
```shell
cp .env.example .env
```
4. start expo
```shell
npx expo start
```
